⚙️ Configure o Firebase
O ADM precisa fazer isso uma única vez. Depois, qualquer músico abre o link e já entra diretamente.
1
Acesse console.firebase.google.com → crie um projeto gratuito
2
Vá em Realtime Database → Criar banco → modo de teste
3
Copie a URL do banco (ex: https://SEU-PROJETO.firebaseio.com)
4
Abra este arquivo HTML e substitua na linha indicada:
const FIREBASE_URL = 'https://SEU-PROJETO-default-rtdb.firebaseio.com';
Feito isso, distribua o arquivo. Qualquer celular que abrir já conecta automaticamente — sem senha, sem código de acesso.
